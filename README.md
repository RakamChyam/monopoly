Многопользовательская игра «Монополия» для локальной сети.
1 сервер = 1 игра, к которой игроки присоединяются.

ИНСТРУКЦИЯ ПО ЗАПУСКУ
1. Убедитесь, что у вас установлен node.js и npm

2. После установки вводите в консоли следующее:
   cd frontend/client
   npm install         
   npm run build        

3. Далее вводим следующее:
   cd ..
   cd ..
   cd backend       
   npm install       
   npm run build    
   npm start         

4. Заходим в браузер и вводим localhost:5000, если вы видите форму входа, то все получилось.

Для игры с друзьями вам нужна общая локальная сеть. Я рекомендую использовать RadminVPN для этого.
Тогда в браузере ваши друзья должны будут вводить следующее: http://[IP-адрес RadminVPN]:5000
Пример: 26.103.105.210:5000
